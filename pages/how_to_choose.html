Для тех, кто не хочет читать, есть краткий курс по выбору на английском

Человек, который стоит на пути выбора, должен определиться не с языком программирования, а с направлением, в котором он хочет работать. Школьникам в этом плане достаточно просто: перед ними не стоит вопрос заработка на написании кода, а в получении базовых знаний о программировании как таковом.

В связи с этим есть три варианта, что можно изучить в первую очередь:

Python C++ Pasсal
Этот язык очень прост в изучении, у него понятный синтаксис, однако перейти с этого языка на C-подобный будет тяжело из-за существенных различий в записи кода и принципов его исполнения. Еще одним минусом является большй объем встроенных функций (сортировка, выбор максимального/минимального элемента и т.п.) Результат - незнание Python-программистов элементарных основ. "Тяжело в учении, легко в бою" - примерно так можно описать изучение плюсов. Синтаксис для человека, только что открывшего среду разработки, невероятно сложный, но, если постараться и вытерпеть, то все старания окупятся. Программистам на C++ неплохо платят, так что хорошее знание этого языка программирования всегда пригодится. Бессмертная классика, помогающая ни одному поколению программистов усвоить основы: инициализация переменных, четкие рамки для циклов и ветвлений. После этого языка программирования будет легко изучить другой язык, будь то питон или плюсы. Основы, закладывающиеся при изучении Pascal'я, применяются при разработки на любом языке программировании.

А теперь вернемся к отдельным направлениям в IT-сфере и к языкам, которые в них используются

WEB-программирование

Веб-разработка представляет собой процесс создания сайтов. Разработка внешнего вида сайта называется фронтенд, а части, работающей на стороне сервера и выполняющей программную часть, - бекенд. Для этих направлений используются свои языки программирования (и разметки).

Фронтенд

Внешний вид сайтов чаще всего создают декларативные языки HTML и CSS (на них нельзя написать полноценную программу, они только определяют расположение элементов на странице). Ну а полноценным языком в фронтенд-разработке считается JavaScript. Он придает динамику страницам, скрипты помещаются в сам докупент страницы и запускаются в браузере, они снижают нагрузку на сервер.

Бекенд

Для разработки бекенда используется сразу несколько современных и высокоуровневых языков:

  • Java
  • Python и специальные фреймворки Django, Pyramid, микрофреймворк flask
  • PHP

Мобильная разработка

Так как в настоящее время существует две наиболее популярные ОС для смартфонов, то разработку на Android и iOS стоит рассмотреть отдельно.

Для разработки на Android используется язык Java, т.к. Google предоставила для своей мобильной ОС инструментарий SDK, базирующийся на этом языке программирования.

Для создания приложений, рассчитанных на работу на iOS, используется язык от компании Apple - Swift и несколько устаревший Object-C.

Создание игр

Немного устаревшие языки C++ и Java отошли на второй план, сейчас в разработке игр используются движки. Наиболее востребованным является Unity.

Разработка десктопных (настольных) приложений

  • C++ и Python используются для разработки в связке с фреймворком Qt.
  • Java располагает собственными возможностями для создания графических интерфейсов (AWT, Swing, JavaFX).
  • Можно даже создавать приложения на JavaScript, используя Electron, Node Webkit.
